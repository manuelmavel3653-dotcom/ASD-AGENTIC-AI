# ASD Agentic AI Prototype

This project demonstrates a simple Agentic AI prototype developed for an Autism Spectrum Disorder (ASD) support system.

## Features

- Declarative Knowledge Rules
- Procedural Tools
- Communication Constraints
- Scenario-Based Testing
- Execution Trace

## Architecture

User Input
↓
Problem Identification
↓
Rule Selection
↓
Tool Activation
↓
Response Generation
↓
Final Output
